React Project

Explore my React project showcasing an array of shopping goods. The application utilizes MongoDB Atlas for data storage, with Express serving as the backend framework to handle HTTP requests and perform CRUD operations on the MongoDB database. User authentication is implemented using JSON Web Tokens (JWT), ensuring secure access to the application and restricting CRUD actions to authenticated users. Users can perform CRUD operations, including creating, reading, updating, and deleting shopping goods data.
